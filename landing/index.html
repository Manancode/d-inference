02 — The Trust Problem
Other decentralized compute networks connect buyers and sellers. That's the easy part.

The hard part: you're sending prompts to a machine you don't own, operated by someone you've never met. Your company's IP. Your users' data. Your competitive advantage — on a stranger's laptop.

Without verifiable privacy, decentralized inference is a non-starter.
03 — The Architecture

Verifiable privacy, not promised privacy

Four independent layers. Each one is verifiable. Together they make it cryptographically impossible for a node operator to access your data.

Encryption

End-to-end encrypted

Your prompt is encrypted with the node's X25519 public key before it leaves your machine. NaCl box. The coordinator routes traffic but cannot read it.

Hardware

Secure Enclave attested

Each node holds a P-256 key pair generated inside Apple's tamper-resistant Secure Enclave. The attestation chain traces back to Apple's Enterprise CA.

Runtime

Hardened process

PT_DENY_ATTACH, binary self-hash, SIP enforcement, Hypervisor.framework stage-2 page tables. The operator cannot attach to or inspect the inference process.

Output

Traceable to hardware

Every response is signed and traceable to the specific verified machine that produced it. The attestation is public — you can verify it yourself.

The operator runs your inference. They cannot see your data.

Prompts are encrypted before they leave your machine. The coordinator routes traffic it cannot read. The provider processes tokens it cannot extract. The entire attestation chain is public — verify it yourself.

04 — For Developers

OpenAI-compatible

Swap your base URL. The rest of your code stays the same — streaming, function calling, all of it.

python 
from openai import OpenAI

client = OpenAI(
    base_url="https://api.darkbloom.ai/v1",
    api_key="your-api-key"
)

response = client.chat.completions.create(
    model="qwen3-30b-a3b",
    messages=[{"role": "user", "content": "Hello!"}],
    stream=True
)

for chunk in response:
    print(chunk.choices[0].delta.content, end="")
